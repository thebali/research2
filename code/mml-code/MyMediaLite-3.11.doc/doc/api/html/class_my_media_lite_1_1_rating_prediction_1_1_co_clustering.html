Detailed Description

Co-clustering for rating prediction

Literature:

  • Thomas George, Srujana Merugu A Scalable Collaborative Filtering Framework based on Co-clustering. ICDM 2005. http://hercules.ece.utexas.edu/~srujana/papers/icdm05.pdf

This recommender does NOT support incremental updates.
