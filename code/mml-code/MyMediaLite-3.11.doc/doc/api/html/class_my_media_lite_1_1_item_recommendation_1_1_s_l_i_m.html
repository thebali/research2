Detailed Description

Abstract class for SLIM based item predictors proposed by Ning and Karypis

This class only implements the prediction model presented in the original paper.

Literature:

  • X. Ning, G. Karypis: Slim: Sparse linear methods for top-n recommender systems. ICDM 2011. http://glaros.dtc.umn.edu/gkhome/fetch/papers/SLIM2011icdm.pdf
