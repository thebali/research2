Detailed Description

SVD++: Matrix factorization that also takes into account what users have rated

Literature:

  • Yehuda Koren: Factorization Meets the Neighborhood: a Multifaceted Collaborative Filtering Model. KDD 2008. http://research.yahoo.com/files/kdd08koren.pdf
